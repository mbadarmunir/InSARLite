The Turkey case study demonstrates:
- Precursory deformation detection before catastrophic landslide failure (December 8, 2024)
- Complete workflow from installation to scientific results
- Processing 60 Sentinel-1 acquisitions
- SBAS time series analysis with mean VLOS velocities up to 25 mm/yr
- Publication-quality results and interpretation
